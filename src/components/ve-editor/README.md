# ve-editor 锚点

## 功能描述

1. 富文本编辑器
2. 预览

---

## 使用说明

https://imzbf.github.io/md-editor-v3/en-US/index

### 安装

```shell
npm install axios

npm install md-editor-v3 --save

npm install ve-editor
```

### 卸载

```shell
npm uninstall axios

npm uninstall md-editor-v3 --save

npm uninstall ve-anchor
```
