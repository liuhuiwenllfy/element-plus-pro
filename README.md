# element-plus-pro

v1.0.0 | 2024-02-02

## 项目概述

基于vue3+element-plus结合开发业务扩展的组件库

---

## 使用说明

1. 安装
    ```shell
    npm install
    ```
2. 运行
    ```shell
    npm run dev
    ```
3. 打包
    ```shell
    npm run build
    ```
